# Mini Docs API

Небольшой учебно-практический API-проект на Node.js и Express.

## Что реализовано
- получение списка документов
- получение документа по id
- создание документа
- обновление документа
- удаление документа
- базовая валидация
- обработка ошибок

## Стек
- Node.js
- Express
- JavaScript

## Запуск проекта
```bash
npm install
node app.js 
```


## Основные маршруты API
- GET /docs
- GET /docs/:id
- POST /docs
- PUT /docs/:id
- DELETE /docs/:id
